# CLI

go build: compiles  
go run: compiles and immediately executes one or two files  
go fmt: formats all the code in current dir  
go install: compiles and "installs" a package  
go get: downloads the raw source code of someone else's packcage  
go test: run tests

# Types of packages

## Executables

`main.go`

## Reusables

used as dependency
